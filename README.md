# Reference


## Population Mean
Calculate the population mean.

StatisticsCalculator.mean(data)
data: a list of numbers.
return: the mean of the input list.




## Median
Calculate the population mean.

StatisticsCalculator.getMedian(data)
data: a list of numbers.
return: the median of the input list.

```
>>> getMedian([0, 3, 5, 5])
4.0
```

```
>>> median([2, 3, 5])
3
```
